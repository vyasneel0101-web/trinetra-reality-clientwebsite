## 📌 Overview

Trinetra Reality needed a modern, conversion-focused website to showcase properties and capture leads online. This is a complete freelance deliverable — a single optimised HTML/CSS/JS file with zero dependencies or build steps required.

Design direction: **"Quiet Luxury"** — deep navy (`#0A1628`) paired with a gold accent system (`#C9A84C`), glassmorphism surfaces, smooth animations, and full dark/light mode support.

---

## ✨ Features

### 🔍 Smart Property Search & Filters
- Hero search bar with glassmorphism design — search by city, area, or landmark
- Category filter buttons to browse by property type
- Advanced filter panel — location, area (sq.ft), price range, furnishing type

### 🏡 Property Listings & Comparison Tool
- Filterable property cards with badges, area tags, and hover effects
- **Property Comparison Tool** — side-by-side comparison of up to 2 properties
- Property detail modal with full specs (area, furnishing, location)

### 🗺️ Area Carousel
- Interactive sliding carousel showcasing key localities in Kutch
- Smooth navigation with prev/next controls

### 🧮 EMI Calculator
- Real-time EMI calculation with animated **donut chart visualisation**
- Inputs: loan amount, tenure, interest rate
- Helps buyers make informed decisions directly on the website

### 📊 Animated Stats Section
- Live counter animations for properties sold, happy clients, years of experience, etc.

### 🤝 Sell Your Property Form
- Dedicated seller section with a lead capture form
- Toast notification on submission — *"Our expert will call you within 2 hours for a free valuation!"*

### 💬 Chatbot
- Built-in floating chatbot button with pulse animation
- Instant engagement for visitor queries

### 📩 Contact & Inquiry Form (EmailJS)
- Full inquiry form integrated with **EmailJS** — no backend needed
- Direct email delivery to client on every lead submission

### 📝 Blog Section
- Articles section for property tips, market insights, and local guides
- Improves SEO and establishes brand authority

### 🧑‍💼 Team / About Section
- Agent/team profiles with role and contact info

### 📸 Instagram Feed Section
- Social proof section linking to the agency's Instagram

### 🍪 Cookie Consent Bar
- GDPR-style cookie consent banner with animated slide-up

### 🚪 Exit Intent Popup
- Captures leaving visitors with a targeted offer popup

### 🌙 Dark / Light Mode Toggle
- Full dark and light theme support via CSS variables
- Smooth transition between themes

### 🔔 Recent Properties Strip
- Fixed top announcement bar showing newly listed properties

### ⚡ Preloader Animation
- Branded preloader on page load for a polished first impression

### 📱 Fully Responsive
- Works seamlessly across mobile, tablet, and desktop

## 🚀 Getting Started

No installation or build step needed.

```bash
# Clone the repo
git clone https://github.com/vyasneel0101-web/trinetra-reality-website.git

# Open directly in browser
open index.html
```

To enable the contact form, update EmailJS credentials in `index.html`:

```javascript
emailjs.init("YOUR_PUBLIC_KEY");
// Update serviceID and templateID with your EmailJS dashboard values
```
